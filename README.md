# 🌦️ Weather App

A modern and responsive Weather Application built using React that provides real-time weather information based on user location or search.

---

## 🚀 Features
- 🌍 Get weather by city name
- 🌡️ Display temperature, weather conditions
- 📍 Location-based weather (optional)
- ⚡ Fast and user-friendly interface
- 📱 Responsive design

---

## 🛠️ Tech Stack
- React
- JavaScript
- HTML
- CSS
- Weather API
